title: "Flask - Debug"
date: 2020-01-07
categories: Flask
---

Turn on Debug mode
```
(venv) (base) ➜  microblog git:(master) ✗ export FLASK_DEBUG=1
(venv) (base) ➜  microblog git:(master) ✗ flask run   
```

Turn off Debug mode
```
(venv) (base) ➜  microblog git:(master) ✗ export FLASK_DEBUG=0  
```
